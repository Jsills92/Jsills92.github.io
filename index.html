Back End Development and APIs Projects (2025)

Timestamp Microservice

The Timestamp Microservice is a web app deployed through Railway that converts a given date string or Unix timestamp into a human-readable UTC format and vice versa.

Request Header Parser Microservice

The Request Header Parser Microservice is a web app deployed through Railway that extracts and returns the client's IP address, preferred language, and user-agent information from the request headers, offering a quick way to access client-specific details in JSON format.

URL Shortener Microservice

The URL Shortener Microservice is a web app deployed through Railway that converts long URLs into shortened links for easy sharing. Users can submit a valid URL, which is stored and assigned a unique short identifier. When accessed, the short URL redirects to the original link, providing a simple and efficient way to manage URLs.
